## 🎨 Diseño y Estilo

### Paleta de Colores
| Uso | Color |
|-----|-------|
| **Brand Teal** | `#75c9be` |
| **Brand Pink** | `#f39dbe` |
| **Beige** | `#F5F3F0` (fondo alternado) |
| **White** | `#ffffff` |
| **Gray 50** | Cajas y fondos neutros |
| **Gray 900** | Textos y bordes principales |
| **Green orgánico** | `#15803D` (botón álbum físico) |
| **WhatsApp Green** | `#25D366` (botón plantilla) |

### Estilo de Diseño
- **Concepto:** Editorial limpio, minimalista, inspirado en sitios de plantas
- **Sin emojis** en botones principales (solo texto limpio)
- **Sin degradados** (excepto fondos muy sutiles)
- **Cajas:** `bg-gray-50` con `rounded-2xl`, sin sombras fuertes
- **Botones:** `rounded-full` con sombras suaves
- **Tipografía:** Geist (sans-serif limpio)
- **Mobile First:** Diseñado para mayoría de usuarios móviles en Colombia

### Componentes Clave
1. **Header:** Sticky con logo, navegación a secciones, Instagram, menú hamburguesa móvil
2. **Hero:** Layout 2 columnas, countdown de 7 días, stats grid, sin emojis
3. **PhysicalAlbum:** Badge "Para Familias", combo en gris, feature cards limpias
4. **DigitalTemplate:** SÚPER BONO destacado, garantía en caja gris con borde
5. **AboutUs:** Copy con tono positivo y pasión (sin lástima), sin galería de taller
6. **Footer:** Centrado en móvil, información simplificada

- **Framework:** Next.js 15 (App Router) + TypeScript
- **Estilos:** Tailwind CSS v4
- **Imágenes:** next/image con WebP
